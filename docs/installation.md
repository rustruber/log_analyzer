# Installation

## Stable release

To install log_analyzer, run this command in your terminal:

```sh
uv add log_analyzer
```

Or if you prefer to use `pip`:

```sh
pip install log_analyzer
```

## From source

The source files for log_analyzer can be downloaded from the [Github repo](https://github.com/rustruber/log_analyzer).

You can either clone the public repository:

```sh
git clone https://github.com/rustruber/log_analyzer
```

Or download the [tarball](https://github.com/rustruber/log_analyzer/tarball/main):

```sh
curl -OJL https://github.com/rustruber/log_analyzer/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd log_analyzer
uv sync
```
